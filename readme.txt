== Description ==
"Multiple email recipient for woo orders" plugin helps Customer to add additional email address, that can be used as extra email recipient for WooCommerce new order.
Most of the corporate customers require their order emails to be sent to the multiple recipient, so that their senior or junior coworker comes to know that a order has been placed for particular product. With the help of this plugin, customer can add additional email id in the checkout form.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the /Plugins/ screen in WordPress
3. No Settings required its plug n play plugin.

== Frequently Asked Questions ==
= How many email recipients can add into this plugin as Bcc? =
Customer can add one additional email id in the checkout form. 
= Is this plugin works currently for particular email notification? =
This plugin send only new order email to additional email id.
